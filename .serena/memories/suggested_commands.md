# 開発コマンド
- `docker compose up` (ローカル開発の起動)
- `cd frontend && npm run dev` (フロント開発)
- `cd frontend && npm run test` (フロントテスト)
- `cd frontend && npm run lint` (フロントLint)
- `cd frontend && npm run build` (フロントビルド)
- `cd backend && bin/rails server` (バックエンド起動)
- `cd backend && bin/rails test` (バックエンドテスト)

# 便利コマンド
- `rg "pattern"` (高速検索)
- `rg --files` (ファイル一覧)
- `ls`, `find`, `git status` など
